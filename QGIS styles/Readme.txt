QSMSidewalkSHP.qml 
This is a simple style file for use with QGIS to show data about pavements ('sidewalks') on roads, as downloaded from Openstreetmap data using the QGIS plugin 'QuickOSM' The relevant QuickOSM query is to download data corresponding to "sidewalk=*"

These are text files to be used to define colours used by the GDAL raster terrain analysis tool - color relief function  
e.g. gdaldem color-relief [output file] [text file] [input file] -of GTiff
These are designed to provide green through to red/brown/yellow shades for Scotland.
